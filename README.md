# task-manager

Sistema de gerenciamento de tarefas desenvolvido em Flask.

## Recursos utilizados:
- SqlAlchemy: ORM para o DB SQLite
- Bootstrap: Armonização visual, responsividade, ícones e uso de cards
- WTForms: Gerenciamento dos formulários
